trim

trims off any white space at the beginning or end of a string useful for input data from user let userInput = " hello" userInput.trim() "hello" doesn't remove other characters we can chain methods together let greeting = " hello" greeting.trim().toUpperCase() "HELLO"

arg

thing.method(arg) some methods accept arguments that modify their behavior think of them as inputs that we can pass in we pass these arguments inside of the parentheses example: let tvShow = 'catdog'; tvShow.indexOf('cat'); // 0 tvShow.indexOf('dog'); // 3 tvShow.indexOf('z'); // -1 (not found) returns first occurence

slice

slice actually can accept more than one argumentslook at the docs extract or slice a portion of a string and gives us a new string str.slice(beginIndex[, endIndex]) endIndex is optional example: let msg = "haha that is so funny!".slice(5); -> 'that is so funny!' msg.slice(5, 9) -> "that" can you negative numbers it just works backwards

replace

need to pass in 2 arguments 1) what we're trying to replace 2) what we want to replace it with msg.replace('haha', 'lololol') -> 'lololol that is so funny!' also a replace all method but not fully implemented these days
